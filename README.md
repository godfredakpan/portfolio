## Quick start

Clone the repo

```bash
  git clone https://github.com/blaiti/Developer-Portfolio.git
```

Install Developer Portfolio with npm

```bash
  cd Developer-Portfolio
  npm install
```

## Run Locally

To run locally, run the following command

```bash
  npm run dev
```

## Deployment

To create a production build

```bash
  npm run build
```
